Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

User Registration

Example request:
curl --request POST \
    "http://localhost/api/signup" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"pbowfvk\",
    \"email\": \"ashlynn.armstrong@example.com\",
    \"password\": \"211P5#~E[y}CwK}\",
    \"confirm_password\": \"deleniti\"
}"

const url = new URL(
    "http://localhost/api/signup"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "pbowfvk",
    "email": "ashlynn.armstrong@example.com",
    "password": "211P5#~E[y}CwK}",
    "confirm_password": "deleniti"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/signup

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

name   string   

Must not be greater than 255 characters. Example: pbowfvk

email   string   

Must be a valid email address. Example: ashlynn.armstrong@example.com

password   string   

Must be at least 6 characters. Must not be greater than 255 characters. Example: 211P5#~E[y}CwK}

confirm_password   string   

The value and password must match. Example: deleniti
